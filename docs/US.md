# US.md — User Stories

## 에픽 1: 회원 인증

### US-01 회원가입
**As a** K-POP 팬  
**I want to** 이메일과 비밀번호로 회원가입하고 싶다  
**So that** 커뮤니티에서 글을 쓰고 소통할 수 있다

**인수조건:**
- Given 미가입 사용자가 /signup 접속 시
- When 이메일, 비밀번호, 닉네임 입력 후 제출하면
- Then 회원가입 완료 후 메인 페이지로 이동한다

### US-02 로그인
**As a** 가입된 사용자  
**I want to** 이메일/비밀번호로 로그인하고 싶다  
**So that** 내 계정으로 활동할 수 있다

**인수조건:**
- Given 로그인 페이지에서 올바른 정보 입력 시 → 로그인 성공
- Given 잘못된 비밀번호 입력 시 → 에러 메시지 표시

---

## 에픽 2: 소통 공간 (자유게시판)

### US-03 게시글 작성
**As a** 로그인한 사용자  
**I want to** 글과 이미지를 올리고 싶다  
**So that** 다른 팬들과 K-POP 정보를 공유할 수 있다

**인수조건:**
- Given 로그인 상태에서 글쓰기 버튼 클릭 시
- When 제목, 내용, 이미지(선택) 입력 후 제출하면
- Then 게시글 목록에 새 글이 표시된다

### US-04 좋아요
**As a** 사용자  
**I want to** 게시글에 좋아요를 누르고 싶다  
**So that** 좋은 글에 공감을 표현할 수 있다

**인수조건:**
- When 좋아요 버튼 클릭 시 → 페이지 새로고침 없이 숫자 변경
- When 이미 누른 좋아요 다시 클릭 시 → 취소

### US-05 댓글/대댓글
**As a** 사용자  
**I want to** 게시글에 댓글을 달고 싶다  
**So that** 작성자와 소통할 수 있다

**인수조건:**
- When 댓글 입력 후 제출 → 목록 하단에 즉시 표시
- When 댓글의 답글 버튼 클릭 → 대댓글 입력창 표시

---

## 에픽 3: 공지사항

### US-06 공지사항 조회
**As a** 사용자  
**I want to** 공지사항을 확인하고 싶다  
**So that** 커뮤니티 규칙과 이벤트를 알 수 있다

**인수조건:**
- 공지사항은 게시판 상단 고정 표시
- 관리자만 작성/수정/삭제 가능

---

## 에픽 4: 주요 일정

### US-07 일정 조회
**As a** K-POP 팬  
**I want to** 아이돌 컴백/콘서트 일정을 보고 싶다  
**So that** 중요한 날짜를 놓치지 않을 수 있다

**인수조건:**
- 캘린더 형태로 월별 일정 표시
- 날짜 클릭 시 해당 날짜 일정 목록 표시

---

## 에픽 5: 투표

### US-08 투표 참여
**As a** 사용자  
**I want to** 팬 투표에 참여하고 싶다  
**So that** 의견을 표현하고 결과를 확인할 수 있다

**인수조건:**
- 투표 항목 선택 후 제출 → 결과 그래프 표시
- 1인 1회만 투표 가능

---

## 에픽 6: 관리자

### US-09 관리자 대시보드
**As a** 관리자  
**I want to** 회원과 게시글을 관리하고 싶다  
**So that** 커뮤니티를 건전하게 운영할 수 있다

**인수조건:**
- Django Admin에서 회원 목록 조회/수정/삭제
- 신고된 게시글 삭제
- 공지사항 작성/수정/삭제
